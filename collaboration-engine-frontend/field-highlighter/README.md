# Field Highlighter

## Setup

```sh
pnpm i
```

## Run demo

```sh
npm start
```

## Run tests

```sh
npm test
```

## Debug tests

```sh
npm run debug
```

See [@web/test-runnner](https://github.com/modernweb-dev/web/tree/master/packages/test-runner) documentation.

## Visual tests

Make sure you export `SAUCE_USERNAME` and `SAUCE_ACCESS_KEY` env variables.

### Run visual tests

```sh
npm run test:visual
```

### Update screenshots

```sh
npm run screenshots
```
